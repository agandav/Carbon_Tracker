# Carbon Tracker (SmartScheduler for Green AI)

This project demonstrates carbon-aware scheduling for AI/batch workloads using:

- Checkpoint 1: Greedy immediate execution baseline
- Checkpoint 2: Batch optimization for lower emissions
- ML demo: Random Forest predictions for savings and delay recommendations

The code is designed to run locally on any computer with Python 3.9+.

## Project Structure

- core/: Scheduling, optimization, carbon forecast utilities, and ML predictor
- demos/: Runnable scripts for checkpoints and ML demo
- results/: JSON outputs generated by runs
- visualizations/: Plotting scripts
- requirements.txt: Python dependencies

##  Run the Project

Run commands from the repository root.

Checkpoint 1 (greedy baseline):

	python demos/test_checkpoint1.py

Checkpoint 2 (batch optimizer + monitor):

	python demos/test_checkpoint2.py

ML demo:

	python demos/demo_ml.py


## Output Files

After running scripts, outputs are saved in results/:

- checkpoint1_results.json
- checkpoint2_results.json
- ml_enhanced_results.json
